# 🌸 花中班聯會官方網站

這是 **花中班聯會** 的官方網站，由學生自行設計與維護，提供即時公告、活動資訊，以及校園相關服務。  
我們希望透過這個平台，讓全校師生能更方便地獲取資訊，並參與更多元的校園活動。

---

## ✨ 網站特色
- 📢 **即時公告**：快速發布班聯會的重要消息  
- 🎉 **活動資訊**：查詢近期及歷年活動詳情  
- 🏫 **校園專區**：介紹公共設施與服務  
- 🛠 **學生權益**：提供檢修與建議管道，保障學生需求
